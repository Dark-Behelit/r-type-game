Generador de Recursos - R-Type Clone

Instrucciones

Este generador creará los recursos temporales necesarios para el juego:

  • Imágenes: fondo espacial, nave del jugador, enemigos, disparos y explosiones
  • Sonidos: efectos de disparo, explosión, game over y música de fondo

Los archivos se guardarán automáticamente en las carpetas correspondientes.
